## 项目定位
参考 [funNLP](https://github.com/fighting41love/funNLP) 的组织方式，做一个**分类清晰、持续更新、中文友好**的人形机器人资料库。

适合人群：
- 想快速入门人形机器人方向的同学
- 想找可复现开源项目的开发者
- 想做产业研究（批发/零售/仓储/快递/运输）的分析者

## 当前收录

- ✅ 论文：30 条（可读版）
- ✅ 开源项目：20 条（可读版）
- ✅ 总计：50 条初版骨架已升级为可读版

重点覆盖方向：
- 运动控制与全身控制
- 灵巧操作与双臂协作
- VLA（视觉-语言-动作）
- 数据与仿真（Sim2Real）
- 人机交互（HRI）
- 产业应用（零售/仓储/物流等）
