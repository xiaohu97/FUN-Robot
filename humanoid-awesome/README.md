> 人形机器人论文与开源项目合集（参考 funNLP 的“分类清晰 + 表格化 + 持续更新”风格）

## 目标
- 做一个中文友好、可维护的人形机器人资源库
- 覆盖：论文、开源项目、数据集、基准、产业落地方向
- 支持“快速扫一眼 + 深入跟踪”两种阅读方式

## 标签规范（建议）
- 任务：`locomotion` `manipulation` `whole-body-control` `imitation-learning` `VLA`
- 场景：`factory` `warehouse` `retail` `delivery` `service`
- 形态：`sim` `sim2real` `real-world`
- 状态：`sota` `classic` `reproducible` `todo`

## 贡献规范（简版）
- 每条至少含：标题、年份、任务标签、链接
- 论文优先补：摘要3行 + 应用价值 + 是否值得细读
- 项目优先补：README质量 + stars + 维护活跃度 + 可复现性
